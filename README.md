Este repositorio contiene el código para la implementación de la técnica de demodulación homodina digitalmente, a partir de la medición de la señal de interés y de la señal de referencia mediante la placa de adquisición Personal DAQ 3001. 

El código "obtener_alturas.py" permite analizar el archivo binario de ejemplo, que viene acompañado de la metadata de la adquisición, utilizada para parsear el mismo. 

Por otro lado, el directorio "Graficador en tiempo real" contiene la aplicación de la ténica en tiempo real, como su nombre sugiere, al tener la placa de adquisición conectada.
